# project structure

- src
  - config (env config arguments)
  - constants (constants arguments)
  - modules
    - [module]
      -  controllers
      -  services
      . `[module].module.ts`  
  - services (public services)
  - types
  . `app.controller.ts`  
  . `app.modules.ts`  
  . `app.service.ts`  
  . `main.ts`
